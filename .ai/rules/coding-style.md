# レビュー指摘から昇格した規約

レビューで指摘された内容を再発防止のため規約化したものです。
`foundruu rules add "<指摘内容>"` で追記されます。実装・レビュー前に必ず読んでください。

- 変数名は省略しない。複数行スコープで生きる変数に1〜2文字名(t, m, nm 等)を使わず、意味のある名前(trace, matched, nodeModulesDir 等)を付ける。1行完結のコールバック引数は慣習的な短縮名でもよい（2026-07-08 追加）
- エクスポートする関数・定数には日本語1行の JSDoc を必ず書く(何を・なぜ)。型で自明でない引数にのみ @param を添える。型情報の繰り返しは書かない（2026-07-08 追加）
